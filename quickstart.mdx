## Get started with Neutral Art

Follow these steps to download, install, and configure Neutral Art with AI assistance.

### Step 1: Download and Install Neutral Art

<AccordionGroup>
  <Accordion icon="download" title="Download Neutral Art">
    Visit the [Neutral Art GitHub releases page](https://github.com/sin-boo) to download the latest version for your operating system (Windows, macOS, or Linux).
    
    Neutral Art is a custom build of Krita, so it will install and run just like regular Krita.
  </Accordion>
  <Accordion icon="circle-play" title="Launch Neutral Art">
    After installation, launch Neutral Art just like you would launch Krita. You'll see the familiar Krita interface with additional AI-powered features in the toolbar or menu.
    
    <Tip>Neutral Art includes all of Krita's features plus AI assistance!</Tip>
  </Accordion>
</AccordionGroup>

### Step 2: Configure your AI API

<AccordionGroup>
  <Accordion icon="key" title="Get your API key">
    Choose one of the supported AI providers and get an API key:
    
    - **Google Studio / Gemini** - Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
    - **OpenAI** - Visit [OpenAI API Keys](https://platform.openai.com/api-keys)
    - **OpenRouter** - Visit [OpenRouter Keys](https://openrouter.ai/keys)
    
    <Note>Future updates will include a Patreon subscription option for access to multiple AI models without needing separate API keys.</Note>
  </Accordion>
  <Accordion icon="gear" title="Enter your API key in Neutral Art">
    1. In Neutral Art, go to **Settings** or the **AI Configuration** panel
    2. Select your AI provider (Google Studio/Gemini, OpenAI, or OpenRouter)
    3. Paste your API key
    4. Click **Save** or **Connect**
    
    <Tip>You can change providers at any time in the settings!</Tip>
  </Accordion>
</AccordionGroup>

### Step 3: Start creating with AI assistance

<AccordionGroup>
  <Accordion icon="image" title="Get feedback on your artwork">
    1. Create or open an artwork in Neutral Art
    2. Select the AI assistant panel
    3. Click **Send Image for Feedback** or similar option
    4. The AI will analyze your artwork and provide constructive criticism and suggestions
    
    <Tip>The AI can help you improve composition, color theory, lighting, and technique!</Tip>
  </Accordion>
  <Accordion icon="comment" title="Ask questions">
    You can also chat with the AI assistant to:
    - Learn how to use Krita features
    - Get creative advice and techniques
    - Ask for help with specific artistic challenges
    - Request tips on improving your workflow
  </Accordion>
</AccordionGroup>
