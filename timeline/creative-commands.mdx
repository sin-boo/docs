## Overview

Creative commands will let you control Krita using natural language instead of hunting through menus or remembering keyboard shortcuts. Just tell the AI what you want, and it happens.

<Note>
  **Status:** Planned for v0.3-0.5 (next 3-6 months)
  
  This feature will work through both text and voice, making it incredibly powerful when combined with voice integration!
</Note>

## How It Works

Instead of navigating menus or memorizing shortcuts, just ask:

```
You: "Lower the opacity to 50%"
AI: ‚úì Layer opacity set to 50%

You: "Create a new layer called 'highlights'"
AI: ‚úì New layer 'highlights' created

You: "Apply a 5 pixel blur"
AI: ‚úì Blur filter applied with 5px radius
```

The AI understands your intent and executes the command in Krita for you.

### Undo Support


### Safe Defaults

- Commands assume safe defaults when parameters aren't specified
- Ambiguous commands prompt for clarification
- Critical actions require explicit confirmation

<Warning>

</Warning>

## Performance Considerations

<AccordionGroup>
<Accordion title="Will commands be instant?">
  Most commands execute nearly instantly. Commands that involve AI interpretation (natural language processing) take 1-2 seconds, while direct commands are immediate.
</Accordion>

<Accordion title="Does this use API credits?">
  Yes, natural language commands use AI tokens to interpret your intent. Simple, direct commands may be optimized to execute locally without API calls in future versions.
</Accordion>

<Accordion title="Can I use this offline?">
  Not initially, as commands require AI interpretation. A future update may add offline support for common commands.
</Accordion>
</AccordionGroup>
