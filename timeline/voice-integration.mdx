## Use Cases

### Getting Real-Time Feedback

```
You: "Hey, I'm working on the lighting for this portrait. 
     Does the rim light on the left feel too strong?"

AI: "The rim light intensity looks good, but consider softening 
    the edge where it meets the shadow. You might also want to 
    add a subtle fill light on the shadow side to maintain 
    facial structure."

You: "How do I soften that edge in Krita?"

AI: "Use a soft round brush with low opacity, set to blend mode. 
    Gently brush along the edge to create a gradual transition..."
```

### Learning While You Work

```
You: "Remind me, what's the rule for color temperature in shadows?"

AI: "Shadows are typically cooler than lit areas. If your light 
    source is warm (like sunlight), your shadows should lean 
    toward cool colors like blues and purples. This creates 
    visual depth and atmospheric perspective."

You: "Got it, thanks!"
```

### Quick Questions

```
You: "What's the shortcut for merging layers?"

AI: "Control+E merges the current layer with the layer below it."

You: "Perfect!"
```
